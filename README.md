# Smart Grid Agentic AI System

This project develops an Agentic AI system for smart grid energy analysis and machine learning-based forecasting. The system allows users to ask natural language questions about energy generation, renewable share, anomalies, trends, and forecasting.

## Features

- Natural language smart grid question answering
- Solar, wind, hydro, coal, gas, nuclear, and total generation analysis
- Date-based energy lookup
- Maximum and minimum generation date detection
- Machine learning-based forecasting
- Anomaly detection using Isolation Forest
- Model comparison using baseline, Linear Regression, and Random Forest
- Streamlit web interface

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Machine Learning
- Smart Grid Energy Dataset

## Machine Learning Models

- Baseline Average Model
- Linear Regression
- Random Forest Regressor
- Isolation Forest for anomaly detection

## Evaluation Metrics

- MAE
- RMSE
- R2 Score
